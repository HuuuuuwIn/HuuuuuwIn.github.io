Spark RDD API

发表于 2015-10-21   |   分类于 spark ， spark基础   |  

为了加深对sparkRDD的理解，写一篇文章讲解下RDD各API的用法。

Actions 解释
map(f: T=>U) RDD[T] => RDD[U]
flatMap(f: T=>Seq[U]) RDD[T] => RDD[U]
filter(f: T=>Bool) RDD[T] => RDD[T]
disdict() RDD[T] => RDD[T]
sample(fract:Float) RDD[T] => RDD[T]
groupByKey() RDD[(K,V)] => RDD[(K,Seq[V])]
combineByKey() RDD[(K,V)] => RDD[(K,Seq[V])]
reduceByKey(f: (V,V)=>V) RDD[(K,V)] => RDD[(K,V)]
union() (RDD[T],RDD[T]) => RDD[T]
join() (RDD[(K,V)],RDD[(K,W)]) => RDD[(K,(V,W))]
cogroup() (RDD[(K,V)],RDD[(K,W)]) => RDD[(K,(Seq[V],Seq[W]))]
mapValue(f: V=>W) RDD[(K,V)] => RDD[(K,W)]
sort(c:Comparator[K]) RDD[(K,V)] => RDD[(K,V)]
cartesian() (RDD[(K,V)],RDD[(X,W)]) => RDD[((K,X),(V,W))]
count() RDD[(K,V)] => Long
collect() RDD[(K,V)] => Seq[T]
reduce(f:(T,T)=>T) RDD[T] => T
lookup(k:K) RDD[(K,V)] => Seq[V]

各个方法的实例如下

map

map执行数据集的转换操作

flatMap

作用同map，不过一个元素可能生成多个结果数据。

filter

过滤操作。

distinct

去重操作。

sample

随机取样本。

第一个参数如果为true,可能会有重复的元素，如果为false，不会有重复的元素；
第二个参数取值为[0,1]，最后的数据个数大约等于第二个参数乘总数；
第三个参数为随机因子。

groupByKey

相同key值集合到一起

combineByKey

对相同的key值的元素执行处理和聚合操作

combineByKey三个参数
createCombiner: V => C,相同key的第一个元素的进入的时候调用
mergeValue: (C, V) => C,相同key的第N个元素（N$\geq$2）进入的时候与第一个函数结果进行合并的函数
mergeCombiners: (C, C) => C，相同key不同分区生成的第二个函数结果的合并函数

reduceByKey

相同的key执行reduce操作

union

两个RDD聚合

join

两个RDD以key聚合
join 包含 join , fullOuterJoin , leftOuterJoin , rightOuterJoin 等多个方法，用法类似于SQL操作。

cogroup

多个RDD聚合

demo结果显示，如果key不匹配会默认给一个空值

mapValue

对key，value结构数据（二元组）的value执行map

sort

RDD排序，sort 提供 sortBy 和sortByKey 等多个方法。

第二个参数为倒序控制

cartesian

cartesian进行笛卡尔积运算

count

count 返回RDD的长度

collect

collect会触发数据的执行，生成结果集

reduce

lookup

lookup查询key为入参的所有数据集
